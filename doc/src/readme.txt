Welcome to Period98
===================
This is (hopefully) the first stable public
release of Period 98.
If you experience any problems, please do not
hesitate to contact the Period98 Homepage

http://dsn.astro.univie.ac.at/period98

If you get an error message concerning a
ctl3d32.dll when starting Period98,
then copy ctl3d32.dll from the ExtraDLL 
subdirectory of the Period98 directory
to the directory c:\windows\system.

In case that there already exists another 
version of ctl3d32.dll, please make a copy 
for safeguarding before installing the
version supplied with Period98.
